## IMPORTANT
cmake is a git submodule
either make submodule.recurse true globally
```
git config --global submodule.recurse true
```
or after cloning this template use...
```
git submodule init
git submodule update
```
or clone it using
```
git clone  --recurse-submodules <link>
```
